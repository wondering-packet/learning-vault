### `git switch -c`

- **Command:**  
  `git switch -c <branch>`

- **Purpose:**  
  Creates and switches to a new branch.

- **Syntax:**  
  ```bash
  git switch -c new-branch
  ```

- **Key Flags:**  
  *(None for basic usage)*

- **Example:**  
  ```bash
  git switch -c feature-x
  ```

- **Related Commands:**  
  - `git branch` – list branches  
  - `git checkout -b` – legacy alternative
