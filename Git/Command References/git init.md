### `git init`

- **Command:**  
  `git init`

- **Purpose:**  
  Initializes a new Git repository in the current directory.

- **Syntax:**  
  ```bash
  git init
  ```

- **Key Flags:**  
  *(None for basic usage)*

- **Example:**  
  ```bash
  git init
  ```

- **Related Commands:**  
  - `git clone` – to copy an existing repo  
  - `git status` – to check repo state
