### `git checkout -b`

- **Command:**  
  `git checkout -b <branch>`

- **Purpose:**  
  Creates and switches to a new branch.

- **Syntax:**  
  ```bash
  git checkout -b new-branch
  ```

- **Key Flags:**  
  *(None for basic usage)*

- **Example:**  
  ```bash
  git checkout -b feature-x
  ```

- **Related Commands:**  
  - `git branch` – list branches  
  - `git switch -c` – modern alternative
