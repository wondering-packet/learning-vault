### `git branch`

- **Command:**  
  `git branch`

- **Purpose:**  
  List, create, or delete branches.

- **Syntax:**  
  ```bash
  git branch [branch-name]
  ```

- **Key Flags:**  
  - `-d`: Delete a branch  
  - `-r`: List remote branches  
  - `-a`: List all branches

- **Example:**  
  ```bash
  git branch
  ```

- **Related Commands:**  
  - `git checkout` – switch branches  
  - `git switch` – modern alternative
