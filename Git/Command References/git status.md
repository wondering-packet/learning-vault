### `git status`

- **Command:**  
  `git status`

- **Purpose:**  
  Shows the working directory and staging area status.

- **Syntax:**  
  ```bash
  git status
  ```

- **Key Flags:**  
  *(None for basic usage)*

- **Example:**  
  ```bash
  git status
  ```

- **Related Commands:**  
  - `git add` – to stage changes  
  - `git diff` – to view unstaged changes
