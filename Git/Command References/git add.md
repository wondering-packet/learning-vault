### `git add`

- **Command:**  
  `git add`

- **Purpose:**  
  Adds file contents to the staging area.

- **Syntax:**  
  ```bash
  git add <file|directory>
  ```

- **Key Flags:**  
  - `-A`: Stage all changes  
  - `.`: Stage everything in the current directory

- **Example:**  
  ```bash
  git add index.html
  ```

- **Related Commands:**  
  - `git commit` – to commit staged changes  
  - `git reset` – to unstage changes
